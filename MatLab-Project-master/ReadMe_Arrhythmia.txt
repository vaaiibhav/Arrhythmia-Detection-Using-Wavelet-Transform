First make sure you have the data file in the same folder than the matlab project is saved to.

Open file ArrhythmicUI



When user presses the RUN button in the editor tab, the user interface is opened



From there the file is first opened which is wanted to be analyzed



Then user should set the appropriate threshold value for peak finder. If peak finder would have been working properly in terms of threshold windowing this step is not needed. 
More about this is explained in the project report.

Then from pop-up menu user is able to choose which plot is wanted to be plotted and analyzed. 



When user pushes ”Do Calculations” button some calculations are made.



Toolbar is located in the upper left corner. From there user can choose zooming and spanning, thus making the analyzing better.
